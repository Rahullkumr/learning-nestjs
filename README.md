## Project setup

```bash
# Install Nestjs globally
npm i -g @nestjs/cli

# Create a new project
nest new project-name
```

## Compile and run the project

```bash
# development
npm run start:dev

# development (20x faster compilation and builds)
npm run start:dev -- -b swc

# production mode
npm run start:prod
```
